# QHM6703_Bayesian
Used for out module Bayesian Statistics, Dataset Proposal.

The source: https://archive.ics.uci.edu/dataset/9/auto+mpg
The original download link is at https://archive.ics.uci.edu/static/public/9/auto+mpg.zip. 

auto-mpg.names describes the relevant information about this data. By deleting rows containing missing values, we obtained auto_mpg_max.csv; further, by retaining only the 'mpg' and 'weight' columns, we obtained auto_mpg_clean.csv.
